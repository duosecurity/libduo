# Installing

Requires SSL development headers (i.e. `libssl-dev`, `openssl-devel`, etc.)

```
$ git clone https://github.com/duosecurity/libduo.git
$ ./configure
$ make
```

# Testing

Auth:

```
$ DUO_API_HOST=<host-here> DUO_IKEY=<ikey-here> DUO_SKEY=<skey-here> ./test-duologin <username>
```
